# Sistema de Control de Ventas y Gastos

Una aplicación web completa para gestionar ventas, gastos y usuarios.

## Características

- ✅ **Login/Logout seguro** con autenticación local
- ✅ **Registro de usuarios** con diferentes roles (Administrador/Empleado)
- ✅ **Registro de ventas** con métodos de pago (Efectivo, Transferencia, Tarjeta)
- ✅ **Registro de gastos** con categorización
- ✅ **Dashboard** con estadísticas en tiempo real
- ✅ **Resumen semanal** de ganancias
- ✅ **Gestión de usuarios** (solo para administradores)
- ✅ **Almacenamiento local** con localStorage
- ✅ **Responsive design** para móviles y escritorio

## Instalación y Uso

1. **Clonar o descargar** los archivos en una carpeta
2. **Abrir `index.html`** en un navegador web moderno
3. **Usar las credenciales por defecto:**
   - Usuario: `admin`
   - Contraseña: `admin123`
